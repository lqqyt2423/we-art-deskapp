# 力气强微信离线助手

### 介绍

微信文章转 PDF 桌面工具，Windows 和 Mac 都可使用。

界面基于 [electron](https://www.electronjs.org/) 开发，PDF 库引用 [wkhtmltopdf](https://github.com/wkhtmltopdf/wkhtmltopdf)。

### 功能特点

- 完美保留微信文章中的图片和样式排版
- 自定义合并多条文章为一个 PDF，且生成的 PDF 包含目录和书签
- 界面交互简单
